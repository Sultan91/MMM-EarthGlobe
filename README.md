# MMM-EarthGlobe
Animated Earth globe module for Magic Mirror
Go to modules folder via console and type <code>git pull https://github.com/Sultan91/MMM-EarthGlobe.git  </code>

Sample config :
<code>
{
	 module:"MMM-EarthGlobe",
	 position:"right",
	   },
</code>
